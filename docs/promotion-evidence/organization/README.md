# Organization Promotion Evidence

This folder contains the first actual promotion candidate execution artifacts for `organization`.

## Artifacts

1. `parity-window-checklist.md`
   - Fill during the minimum 24-hour parity window before policy promotion.
2. `promotion-pr-checklist.md`
   - Fill as the promotion PR approval checklist.
3. `rollback-drill-evidence.md`
   - Generated by the rollback drill command bundle.

Templates:

1. `parity-window-checklist.template.md`
2. `promotion-pr-checklist.template.md`

## Commands

1. Generate rollback drill evidence:
   - `pnpm promotion:organization:rollback-drill`
2. Generate placeholder rollback artifact without running checks:
   - `pnpm promotion:organization:rollback-drill:skip-run`
3. Validate promotion checklist automation:
   - `pnpm ci:promotion:organization:checklist`

## Automation Behavior

- The checklist automation script only enforces checked artifacts when a PR introduces an organization policy promotion to `event-only` in `packages/db/src/truth-compiler/truth-config.ts`.
- On promotion PRs, automation requires:
  1. `promotion-pr-checklist.md` with no unchecked items
  2. `parity-window-checklist.md` with no unchecked items
  3. `rollback-drill-evidence.md` present
